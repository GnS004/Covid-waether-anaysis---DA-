## üîç Analysis Workflow

1. **Data Collection**
   - COVID-19 dataset containing cases, deaths, recoveries, etc.
   - Weather dataset covering temperatures, humidity, wind speed, etc.

2. **Data Cleaning**
   - Handling missing values and inconsistent records.
   - Formatting dates and merging datasets on common keys.

3. **Exploratory Data Analysis**
   - Statistical summaries (mean, median, trends).
   - Time series visualization of cases vs. weather conditions.

4. **Visualization**
   - Heatmaps showing correlation between features.
   - Trend plots over time (cases vs. temperature/humidity).
   - Bar charts and histograms for comparative analysis.
